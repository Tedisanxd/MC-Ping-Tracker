# MC Ping Tracker

A lightweight background app for monitoring Minecraft server latency — built with Python and tkinter.

## Features
- 📡 Real-time ping graph with live updates
- ⚡ Lag spike detection — by absolute threshold or sudden delta jumps
- 🔔 Desktop notifications when ping exceeds your limit
- 🎮 Discord webhook alerts for spikes, server offline, and server online events
- 🎨 5 built-in themes — Dark, Light, Crimson, Midnight, and System (follows Windows)
- 🖥️ Runs silently in the system tray — close the window and it keeps monitoring
- 📊 Session stats — min / avg / max ping and spike count
- ⚙️ All settings saved automatically between sessions

## Requirements
Python 3.9+ and:
pip install mcstatus pystray Pillow plyer

## Usage
Run install.bat once, then launch with run.bat (no console window).
